Código fonte e Pré formatado

o comando document.getElementById('teste") é escrito em linguagem Javascript(JS)

Viu como a formatação do comando está diferente? É assim que a tag del funciona deixa a fonte monoespaçada

Exemplo de código com identação (linguagem phyton)

        
            num = int(imput('Digite um número'))
            if num % 2 = 0
                print (f ' o número {num} é PAR')
            else
                print (f ' o número {num} é IMPAR')
            print ('Fim do programa')

Percebe agora como o exercicio as letras estão em formato monoespaçado, elas são escirtas assim quando se está fazendo código. Porém se você apenas colocar a tag code vai ficar tudo em uma linha só,
ex:

num = int(imput('Digite um número')) if num % 2 = 0 print (f ' o número {num} é PAR') else print (f ' o número {num} é IMPAR') print ('Fim do programa')

para deixar bonitinho a identação que nem no primeiro exemplo você precisa envelopar a tag code com outra tag que é a pre pois essa tag considera todos os espaços e quebras de linha que são feitas dentro dela

Se quiser fazer com que o exemplo do código não fique tão pra frente como está ai é só você selecionar todo ele sem as tags e aperte as teclas shift e tab até colocar na posição desejada. Se por algum acaso queira que o código vá mais pra frente é só apertar tab, por exemplo:

        
        num = int(imput('Digite um número'))
        if num % 2 = 0
            print (f ' o número {num} é PAR')
        else
            print (f ' o número {num} é IMPAR')
        print ('Fim do programa')
