Principais formatações

Negrito / Destaque

Nesse parágrafo está o negrito usando a tag Strong

Itálico / Ênfase

Nesse parágrafo está o Itálico usando a tag em

São os jeitos recomendados para fazer as formatações

Texto marcado

Podemos criar também um texto marcado usando a tag Mark

Nesse parágrafo o texto marcado está em outra cor

Texto grande e pequeno

Para a letra ficar pequena é preciso colocar a tag small

Para deixar a letra grande é apenas no CSS

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, porém não considerado, o famoso texto com risco no meio com a tg del

Texto Incluído

Podemos marcar um texto como sublinhado usando a tag ins. Serve para dar enfase em algo que foi colocado depois

Texto sobrescrito

Para escrever coisas do tipo x20 + 3 use a tag sup, ai o 20 fica pequeno e acima

Texto subscrito

para escrever coisas como H2 O use a tag sub, ai o 2 fica pequeno e embaixo
